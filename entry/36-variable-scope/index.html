36. Java 変数のスコープ

変数のスコープをできる限り狭く保つことで、見通しが良く、処理を追いやすく、バグの少ないプログラムになることを知る

学習

変数のスコープを知る

変数のスコープとは何かを Web で調べてください。検索キーワードは、「 java 変数 スコープ 」です。

また、変数のスコープは、なるべく狭くするのが良いです。その理由についても、いくつか記事を読んで調べてください。検索ワードは、「 java 変数 スコープ 最小限 」です。

練習問題: スコープの修正（時間: 20 分）

次のクラス MoreNarrowScope で使われている変数について、できる限りスコープを狭くするように修正しなさい

修正対象のコード

import java.util.Random;


public class MoreNarrowScope {
    public static void main(String[] args) {


        Random rand = new Random();
        int num = 0;


        int count = 0;
        boolean isHit = false;


        while (!isHit) {
            num = rand.nextInt(10);
            count++;
            System.out.printf("%d 回目の数字 => %d %n", count, num);


            if (num == 5) isHit = true;
        }
    }
}

MoreNarrowScope の実行結果

1 回目の数字 => 8
2 回目の数字 => 8
3 回目の数字 => 2
4 回目の数字 => 4
5 回目の数字 => 6
6 回目の数字 => 0
7 回目の数字 => 5
