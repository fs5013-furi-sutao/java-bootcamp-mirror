Java Bootcamp 学習カリキュラムへようこそ

およそ 250 時間で Java 言語とプログラミングの 基礎 ～ 発展的内容までの知識と技能を手に入れるフリースタイル ITS 所属の方向けのプログラムです

Java の学習をはじめる

明確なゴール設定と PLF

各単元で習得すべきことを明確にしているため、必要な技術や考え方を取りこぼすことがありません。

また、プロジェクトフォーカスラーニング（PFL）によりスキルを身につけるカリキュラム構成になっています。

PFL は数日間で終わる課題(プロジェクト)をカリキュラムの中心に据え、 実践的なアウトプットを繰り返し行うトレーニング手法です。

段階的にレベルアップ

学習内容を「入門編」「 初級編」「中級編」 「 上級編 」「マスタ編」の 5段階に分けているため、 簡単な内容からはじめて、無理なく段階的に学習内容をステップアップしていけます。

豊富な練習問題

練習問題として、100問以上の練習問題を用意しています。 小さな課題に対して、コーディング練習をしていくことで、 コーディングの習慣と問題解決能力を自然と身に付けていけます。

ゲーム作成課題

「入門編」「初級編」「 中級編」 「上級編」「 マスタ編 」の各レベルでゲーム作成課題を用意しています。

なじみのあるゲームを題材にすることで、課題で求められる仕様を理解し、実装するという力が無理なく身に付き、 楽しく試行錯誤してコードをより良くしていく力も身に付きます。

コードレビューできる環境

コードレビューは大変手間の掛かる作業ですが、FreeStyle の学習カリキュラムでは、 コードレビューを何度でも無料で受けることができます。

独学では自分に足りない点や間違っている点に気づけないですが、 コードレビューの機会を利用することで飛躍的に成長することが可能です。

いつでも質問できる環境

いつでも何回でも質問できる体制を整えています。

必要なタイミングで効果的に質問する能力は、業務では必須ですが、 学習カリキュラムを通して、その能力を鍛えることができます。

開発環境の構築にも困らない

Java の開発環境には何が必要で、どうすれば良いのかという初歩的なところから、 最短で効率的に環境を整備、運用するベストプラクティスまでをカリキュラム内で説明しています。

開発環境の構築は慣れていないと、つまづきやすく、時間が掛かるものですが、 質問してもらえれば、いつでもサポートできる体制を整えています。

ダークモード採用

目に優しいダークモードを搭載しています。

学習で長時間、画面を見ていると目に負担が掛かりますが、 画面をダークモードにすることでその刺激を低減することができます。

画面ヘッダにあるアイコンで「ライトモード」「ダークモード」の切り替えが可能です。

  • Java 開発環境（カリキュラム内で手順を説明しています）
